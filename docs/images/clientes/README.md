# Guia rapido - logos de clientes

Esta pasta alimenta automaticamente a secao de clientes no site.

## Como atualizar

1. Coloque os logos nesta pasta: `docs/images/clientes/`
2. Use nomes no padrao:
   - `cliente-01.svg`
   - `cliente-02.svg`
   - `cliente-03.svg`
3. Extensoes suportadas: `.svg`, `.webp`, `.png`, `.jpg`, `.jpeg`
4. Para remover um logo, apague o arquivo.

## Regras

- Sempre use 2 digitos (`01`, `02`, `03`...)
- O site procura automaticamente de `cliente-01` ate `cliente-40`
- Pode haver "buracos" na sequencia e o site continua funcionando

## Recomendacao

- Prefira `.svg` para logos (mais nitido e leve)
